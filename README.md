### Éclat

Immersed in peace and a muted color scheme.

Éclat is a Zed editor theme featuring a smooth, gentle, and peaceful visual style, optimized for long reading sessions and an immersive experience.

Éclat is designed to pair seamlessly with macOS Night Shift and other eye-protection software.
